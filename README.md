# NGdata E-postgenerator

En enkel og stilren webapp for a lage standardiserte e-postutkast til brukerstotte.

## Hva appen gjor

Appen hjelper deg med a:
- Lage ferdige e-postmaler for NANO, Workday og User-S.
- Kopiere utkast rett til utklippstavlen.
- Bruke felles avslutningstekst raskt via egen knapp.
- Velge dagsmodus eller nattmodus.

## Innhold i appen

- NANO
  - Innlogging
  - Reaktivering
- Workday
  - Ansatt bruker
  - Kandidat-profil
- User-S
  - NGFLYT (samme/nytt passord)
  - Ikke NGdata-sak (henvisning til leverandor)

## Bruk

1. Apne [index.html](index.html) i nettleser.
2. Velg sakstype.
3. Fyll inn felter og trykk "Lag utkast".
4. Trykk "Kopier utkast" for a kopiere teksten.

## Teknisk

Prosjektet er en ren frontend-webapp:
- [index.html](index.html)
- [app.js](app.js)
- [styles.css](styles.css)

Ingen backend og ingen byggesteg er nodvendig.

## Formal

Gi 1. linje brukerstotte et raskt og konsekvent verktoy for e-postsvar.
